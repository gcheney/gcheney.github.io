Web Applications

Coder Tutorials

A web app for creating and commenting on coding tutorials

Coder Tutorials is a platform for providing tutorials on coding and software development to others using Markdown. It was built using Node, Express, and MongoDB.

Live Demo Github

Features

  • Local user authentication
  • Authentication through Github using OAuth 2.0
  • User accounts with details and password updates
  • Ability to Create, Read, Update and Delete tutorials using Markdown
  • Ability to Create, Read, Update and Delete reviews of tutorials using Markdown
  • Ability to publish and unpublish tutorials
  • Search by title, content , and author
  • Search only specific users tutorials by title and content

Image Gallery

A web app for uploading and commenting on beautiful images

The Image Gallery is a SPA app built with an Angular front-end that works with a Node/Express API connected to a MongoDB database. It allows users to sign up, comment, and like other users linked images. Image uploading is a planned feature.

Live Demo Github

Features

  • Local user authentication
  • User accounts with overview pages
  • Ability for users to create, view, edit and delete images their linked images
  • Ability for users to to create, view, edit and delete comments on images
  • Search by image title
  • Search only specific users images

Chattrbox

A web app for real-time chat room messaging

Chattrbox is a chat room app built on Socket.io, jQuery, Express and Node. It allows users to choose a screen name and chat room, then dynamically adds them into that room or creates it if it does not exist. No login or registration is required.

Live Demo Github

Features

  • Real time, dynamically created chat rooms
  • Full current users list

Rock, Paper, Scissors

A desktop version of "Rock, Paper, Scissors".

Rock-paper-scissors is a zero-sum hand game usually played between two people, in which each player simultaneously forms one of three shapes with an outstretched hand. These shapes are "rock" (a fist), "paper" (a flat hand), and "scissors" (a fist with the index and middle fingers together forming a V). In this version, the player plays against a virtual ooponent that makes a random selection from the three playabale shapes.

Download  Github 

Spy Games

A desktop guessing game meant to be played with multiple friends.

This game involves a number of players. Each player is given a random word, but one player is given a synonym of that word instead.The players must then ask each other questions related to their word, and at the end of the round make a guess as to which player is "The Spy". The game ends and the players win if they correctly guess the Spy. If the Spy survives all round, then they are the only winner.

Command Line Projects

Sorting Algorithm Timer

A command line application for timing different sorting algorithms.

This project is a command-line utility for comparing the speed of popular sorting algorithms. This could also be used as a sorting algorithm utility library. The user is prompted to select a number, which determines the algorithm that will be used to sort a very large array of numbers. The user can also select the size of the array to be sorted and the range of numbers to include. The utility times this process, determines if the array is indeed sorted, and outputs the result to the console. It also prints the sorted result to a standard text file.
